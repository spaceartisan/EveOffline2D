# EVE Offline 2D

A vanilla JavaScript homage to EVE Online featuring true EVE-like mechanics:

## Features

**Combat System**
- Shield/Armor/Hull damage layers
- Target selection and lock
- Orbit and approach commands
- Capacitor management for weapons

**Mining & Economy**
- Mining lasers with yield upgrades
- Dynamic market prices per system
- Trading ore for ISK
- Cargo hold management

**Multiple Systems**
- High-sec (1.0) - Jita trading hub
- Low-sec (0.3) - Tama danger zone  
- Null-sec (0.0) - NOL-M9 lawless space
- Jump gates connecting systems

**Ship Systems**
- Momentum-based flight physics
- Shield regeneration
- Capacitor regeneration
- Ship fitting and upgrades

## Controls

- **W/S** - Thrust forward/backward
- **A/D** - Turn left/right
- **Space** - Fire weapons (consumes capacitor)
- **M** - Activate mining lasers
- **Click** - Select targets (NPCs/asteroids)
- **Orbit/Approach buttons** - Automated piloting

## How to Play

1. Open [index.html](index.html) in a browser
2. Start in Jita (high-sec) - mine asteroids or trade
3. Upgrade your ship with ISK from mining/bounties
4. Jump to low-sec and null-sec for better rewards (more danger!)
5. Use orbit commands to kite NPCs while shooting

## Technical Details

Pure vanilla JavaScript with Canvas 2D rendering. No frameworks, no dependencies.

Files: [index.html](index.html), [style.css](style.css), [main.js](main.js)
